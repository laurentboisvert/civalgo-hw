# Civalgo - Check-in app

## Installation

1. You need to install dependencies first:

    ```bash
    cd checkin-app
    npm install
    ```

1. Create the database:

    ```bash
    npm run setup-db
    ```

## Usage

```bash
cd checkin-app
npm run dev
```
