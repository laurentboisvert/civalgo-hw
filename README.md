# Civalgo - Check-in app

## Installation

1. You need to install dependencies first:

    ```bash
    cd checkin-app
    npm install
    ```

1. Create the database:

    ```bash
    npm run setup-db
    ```

1. You are going to need `sqlite3` on your system:

    Ubuntu-based systems:

    ```bash
    sudo apt-get install sqlite3
    ```

## Usage

```bash
cd checkin-app
npm run dev
```
